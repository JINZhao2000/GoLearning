# Golang

## 1. Go 语言可以做什么

- 区块链研发工程师
- Go 服务器端
- 游戏、软件工程师
- Golang 分布式
- 云计算

## 2. Go 语言的特点

- 既有静态语言的安全和性能，又有动态语言的开发维护的高效率
- 保留了 C 语言的编译方式和弱化指针
- 垃圾回收
- 天然并发
    - 从语言层面支持
    - goroutine 协程
    - 基于 CPS（Communicating Sequential Processes）并发模型
- 管道通信机制（通过 Channel 进行通信）
- 支持返回多个值
- 切片 slice，延迟执行 defer
